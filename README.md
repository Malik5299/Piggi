# Piggy

Piggy is a smart, AI-driven saving Assistant that runs entirely in your browser.

### Features

**PiggiAI:** Control your goals by natural language inputs via your Gemini API key <br>
**Manual Control:** Add, edit or delete goals manually

### Privacy

**Local Storage:** All your data is saved directly in your browser and never leaves your device <br>
**Secure Key Management:** Your API key is safly saved in the local storage
